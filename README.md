# Snake GAME

One remaster of classic Snake Game make on Python

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install requirements.txt

```bash
pip install requirements    
```

## Usage

```python
import pygame_menu
import pygame
import time
import random
```
